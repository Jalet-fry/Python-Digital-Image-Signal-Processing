# Полный перечень заданий и параметров (Бригада №10, Вариант №10)

## 📐 Общие параметры синтеза сигналов
**Формула:** $s(t) = \sum_{i=0}^{n} A_i \cdot \sin(2\pi \cdot h_i \cdot f_0 \cdot t + \phi)$
*   $t = 0, \frac{1}{f_s}, \frac{2}{f_s}, \dots, \text{duration}$
*   $f_s$ (Sample Rate) = 8000 Гц (для расчетов), 44100 Гц (для аудио)

## 🔊 Лабораторная работа №2: Цифровая фильтрация

### Исходные данные (Вариант 10)
*   **Инструмент:** Виолончель (110 Гц).

| Тип фильтра | Характеристика | Параметры |
| :--- | :--- | :--- |
| **Однородный** | Рекурсивный | $M = 79$ |
| **КИХ (FIR)** | Полосовой, Окно Блэкмана | 80–300 Гц, $M = 151$ |
| **БИХ (IIR)** | Полосовой (Биквадрат) | $f_0 = 200$ Гц, $BW = 60$ Гц |

### Задачи
1.  Реализовать фильтры самостоятельно.
2.  Добавить шум к сигналу и отфильтровать его.
3.  Построить графики сигнала и спектра (в дБ) "До" и "После".
4.  Рассчитать SNR (отношение сигнал/шум) для каждого этапа.
5.  Сравнить эффективность подавления помех.
