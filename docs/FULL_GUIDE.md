## 🛠 1. МАТЕМАТИЧЕСКОЕ ЯДРО (BACKEND)

| Операция | Математическая формула | Код Python (папка `core/`) |
| :--- | :--- | :--- |
| **Синтез** | $\Large s(t) = \sum_{i=0}^{n} A_i \sin(2\pi h_i f_0 t + \phi)$ | `signal += a * np.sin(2 * np.pi * (f0 * h) * t + phases)` |
| **DFT (Прямое)** | $\Large X[k] = \sum_{n=0}^{N-1} x[n] e^{-j \frac{2\pi}{N} kn}$ | `X[k] = sum(x * np.exp(-2j * np.pi * k * n / N))` |
| **IDFT (Обратное)** | $\Large x[n] = \frac{1}{N} \sum_{k=0}^{N-1} X[k] e^{j \frac{2\pi}{N} kn}$ | `x[n] = (1/N) * sum(X * np.exp(2j * np.pi * k * n / N))` |
| **Свертка** | $\Large (x * y)[n] = \sum_{k} x[k] y[n-k]$ | `for k in range(Nx): z[n] += x[k] * y[n-k]` |
| **Теорема** | $\Large x * y = \operatorname{IFFT}(\operatorname{FFT}(x) \cdot \operatorname{FFT}(y))$ | `ifft(fft(x) * fft(y))` |

## 📊 2. ВЕРИФИКАЦИЯ (КАКИЕ ГРАФИКИ СОВПАДАЮТ)

В программе 24 графика разделены на 8 групп. Файлы в `results/plots` именуются по этим группам.

### Группы 1-2: Восстановление (Waveform)
*   **Группа 1 (X):** `[01, 05, 08]` | **Группа 2 (Y):** `[02, 11, 14]`
*   **Смысл:** Оригинал $\approx$ IDFT $\approx$ IFFT.
*   **Защита:** "Сигнал полностью восстановился после обратного преобразования Фурье, значит ошибки данных нет".

### Группы 3-6: Спектры (Spectrum)
*   **Амплитуда X:** `[03, 06, 19]` | **Фаза X:** `[04, 07, 20]`
*   **Амплитуда Y:** `[09, 12, 21]` | **Фаза Y:** `[10, 13, 22]`
*   **Смысл:** Ручное ДПФ $\approx$ БПФ $\approx$ Библиотека.
*   **Защита:** "Все методы дали идентичные пики на гармониках 110, 220, 330, 440 Гц".

### Группы 7-8: Теоремы ЦОС
*   **Свертка:** `[15, 16, 23]` | **Корреляция:** `[17, 18, 24]`
*   **Смысл:** Время (циклы) $\approx$ Частота (FFT).
*   **Защита:** "Перемножение спектров в частотной области полностью заменяет сложную свертку во временной".

## 🔬 3. ЛОГИКА "ЧИСТКИ" И "ПОДГОНКИ"

1.  **Фильтр шума ($\epsilon$):**
    *   `ph[mag < 0.001 * max] = 0`
    *   **Зачем:** Чтобы убрать хаотичные линии фазы на пустых частотах. Мы видим фазу только там, где есть звук.
2.  **Масштаб (Zoom):**
    *   Ось частот обрезана на **600 Гц**.
    *   **Зачем:** Чтобы гармоники инструмента занимали всё поле графика, а не сжимались в углу.

## 📖 4. ТАБЛИЦА ОБОЗНАЧЕНИЙ ДЛЯ ВОПРОСОВ

| Символ | Имя | Что это? |
| :--- | :--- | :--- |
| **$k$** | Bin Index | Номер столбика на графике спектра (частота). |
| **$n$** | Time Index | Номер точки на графике волны (время). |
| **$N$** | Window Size | Количество точек для расчета (1024). |
| **$W_N$** | Twiddle Factor | Комплексный множитель поворота ("Бабочка"). |
| **$\epsilon$** | Epsilon | Погрешность $10^{-13}$ (Машинный ноль). |
