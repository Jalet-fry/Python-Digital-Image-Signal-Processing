## 2. Математическое Ядро (Backend)

| Операция | Математика ($\sum$) | Код Python |
| :--- | :--- | :--- |
| **Синтез** | $\Large s(t) = \sum A_i \sin(2\pi f_0 h_i t)$ | `sum(a * np.sin(2 * np.pi * (f0 * h) * t))` |
| **DFT** | $\Large X[k] = \sum x[n] e^{-j \frac{2\pi}{N} kn}$ | `sum(x * np.exp(-2j * np.pi * k * n / N))` |
| **IDFT** | $\Large x[n] = \frac{1}{N} \sum X[k] e^{j \frac{2\pi}{N} kn}$ | `(1/N) * sum(X * np.exp(2j * np.pi * k * n / N))` |
| **Свертка** | $\Large (x * y)[n] = \sum x[k] y[n-k]$ | `for k: z[n] += x[k] * y[n-k]` |
| **Теорема** | $\Large x * y = \operatorname{IFFT}(\operatorname{FFT}(x) \cdot \operatorname{FFT}(y))$ | `ifft(fft(x) * fft(y))` |
