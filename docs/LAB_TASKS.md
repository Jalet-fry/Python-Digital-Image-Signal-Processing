# Техническое задание: Лабораторная работа №1

## 1. Цель работы
Приобретение навыков программной реализации и верификации алгоритмов ДПФ, БПФ, свертки и корреляции.

## 2. Исходные данные (Вариант №10)
**Формула сигнала:**
$$\Large s(t) = \sum_{i=0}^{n} A_i \cdot \sin(2\pi \cdot h_i \cdot f_0 \cdot t + \phi)$$

| Параметр | x(t) (Виолончель) | y(t) (Контрабас) |
| :--- | :--- | :--- |
| **Амплитуды (A)** | `[1.0, 0.6, 0.4, 0.2]` | `[1.0, 0.7, 0.5]` |
| **Частота (f0)** | `110 Гц` | `55 Гц` |
| **Гармоники (h)** | `[1, 2, 3, 4]` | `[1, 2, 3]` |
| **Фаза (phi)** | `0` | `0` |

**Параметры дискретизации:**
*   Количество отсчетов ($N$): `1024`
*   Частота дискретизации ($f_s$): `8000 Гц`

## 3. Основные задачи
1. Реализовать **линейную свертку** (циклы и БПФ).
2. Реализовать **взаимную корреляцию** (циклы и БПФ).
3. Выполнить прямое и обратное преобразование Фурье (ДПФ и БПФ) для сигналов и результата свертки.
4. Экспериментально подтвердить **Теорему о свертке** (Рис. 16 методички) и **Теорему о корреляции** (Рис. 15 методички).
5. Сравнить самописные алгоритмы с библиотекой **NumPy**.

## 4. Список из 24 обязательных графиков
