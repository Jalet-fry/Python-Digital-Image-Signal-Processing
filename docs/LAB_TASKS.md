# Техническое задание по ЦОСиИ (Вариант №10)

## 🎹 Лабораторная работа №1: Синтез и Фурье-анализ

### 1. Исходные данные
**Формула сигнала:**
$$\Large s(t) = \sum_{i=0}^{n} A_i \cdot \sin(2\pi \cdot h_i \cdot f_0 \cdot t + \phi)$$

| Параметр | x(t) (Виолончель) | y(t) (Контрабас) |
| :--- | :--- | :--- |
| **Амплитуды (A)** | `[1.0, 0.6, 0.4, 0.2]` | `[1.0, 0.7, 0.5]` |
| **Частота (f0)** | `110 Гц` | `55 Гц` |
| **Гармоники (h)** | `[1, 2, 3, 4]` | `[1, 2, 3]` |
| **Фаза (phi)** | `0` | `0` |

*   **N:** 1024 отсчета.
*   **fs:** 8000 Гц.

### 2. Задачи
*   Реализовать **линейную свертку** и **корреляцию** (циклы + БПФ).
*   Реализовать **ДПФ** и **БПФ** (прямое и обратное).
*   Сравнить результаты с библиотекой **NumPy**.
*   Верифицировать теоремы о свертке и корреляции.
