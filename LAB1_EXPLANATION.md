## 1. Математика vs Код (Backend)

| Символ | Значение | Код Python |
| :--- | :--- | :--- |
| $\Large \sum$ | Сумма элементов | `np.sum()` или цикл `for` |
| $\Large X[k]$ | Спектр (частота $k$) | `complex_array` |
| $\Large e^{-j...}$ | Поворачивающий множитель | `np.exp(-2j * np.pi * ...)` |
| $\Large N$ | Длина сигнала (1024) | `len(x)` |

### Основные формулы:
*   **ДПФ (DFT):**
    $$\Large X[k] = \sum_{n=0}^{N-1} x[n] \cdot e^{-j \frac{2\pi}{N} kn}$$
    *Код:* `X[k] = sum(x * np.exp(-2j * np.pi * k * n / N))`
*   **Свертка (Convolution):**
    $$\Large (x * y)[n] = \sum_{k} x[k] \cdot y[n-k]$$
    *Код:* `z[n] += x[k] * y[n-k]`

## 2. Карта совпадений (8 групп по 3 графика)

Графики в папке `results/plots` и в UI сгруппированы так, чтобы доказать точность расчетов через их идентичность.

### 🧬 Группы 1-2: Обратимость Фурье
*   **Группа 1 (X) и Группа 2 (Y)**
*   **Смысл:** Оригинал $\approx$ IDFT $\approx$ IFFT. 
*   **Почему совпадает:** Мы разбираем звук на "ноты" и собираем обратно. Если формулы верны, волна не меняется.

### 📊 Группы 3 и 5: Амплитуды ( DFT $\approx$ FFT $\approx$ Lib )
*   **Группа 3 (X) и Группа 5 (Y)**
*   **Смысл:** Поиск гармоник инструмента.
*   **Почему совпадает:** Это три разных пути к одному результату. БПФ (FFT) — это оптимизация суммы, а не другое преобразование.

### 📐 Группы 4 и 6: Фазы ( DFT $\approx$ FFT $\approx$ Lib )
*   **Группа 4 (X) и Группа 6 (Y)**
*   **Смысл:** Углы сдвига волн.
*   **Очистка:** Если амплитуда $< 0.1\%$, фаза обнуляется, чтобы убрать шум квантования.

### 🔄 Группы 7-8: Теоремы Свертки и Корреляции
*   **Группа 7 (Свертка) и Группа 8 (Корреляция)**
*   **Смысл:** Переход из времени в частоту.
*   **Почему совпадает:** Доказываем, что умножение спектров ($FFT \times FFT$) — это то же самое, что вложенные циклы во временной области.
